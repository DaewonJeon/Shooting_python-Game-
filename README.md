# Shooting_python-Game-

주요 기능 및 업데이트 (Update Log)

✅✅✅ 생존 및 전투 시스템 (Core Gameplay)

생명 시스템 (Life System): 플레이어에게 초기 목숨 3개가 주어지며, 피격 시 1개씩 감소합니다.

무적 시간 (Invincibility): 피격 직후 1초간 무적 상태가 되어 연속적인 죽음을 방지합니다. (캐릭터 깜빡임 효과)

적 AI 개선: 적이 플레이어를 똑똑하게 추적하며, 층(Layer)이 다를 때 멈추지 않고 계속 이동합니다.

적 겹침 방지 (Separation): 적들이 한곳에 뭉치지 않고 서로 밀어내는 물리 로직이 적용되었습니다.


⏸⏸⏸ 편의 기능 (User Interface)

일시정지 (Pause): 게임 중 ESC 키를 누르면 게임이 멈추고 메뉴가 팝업됩니다.

옵션 (Option): 배경음악(BGM)의 볼륨을 조절할 수 있는 기능이 추가되었습니다. (ESC 메뉴에서도 조절 가능)

HUD 개선: 좌측 상단에 하트 아이콘으로 남은 목숨을 직관적으로 보여줍니다.


💎💎💎 아이템 시스템 (Item System)

하늘에서 랜덤하게 아이템이 떨어지며, 획득 시 특수 능력을 얻습니다.

❤️ 하트 (Heart): 목숨을 1개 회복합니다. (최대 3개 유지, 초과 시 증가하지 않음)

🛡️ 쉴드 (Shield): 1회 피격을 막아주는 보호막을 생성합니다. (파란색 오라 이펙트)

👟 스피드 (Speed): 5초간 이동 속도가 1.5배 빨라집니다.


🛠 설치 및 실행 (How to Run)

Python 3.x 설치
pip install pygame 명령어로 라이브러리 설치
main.py 실행
